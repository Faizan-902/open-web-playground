Responsive Navigation Menu

This is a responsive navigation menu that adapts to different screen sizes.

On desktop, the menu items are displayed horizontally. On mobile devices (screen width less than 768px), the menu transforms into a hamburger menu.

Try it out: Resize your browser window or view this on a mobile device to see the responsive behavior!
